Welcome to the SocialZap Use Case Testing Page

Test Case 1:

See Selenium source

This test case tests whether the two links (index.php, video.php) are loaded succesfully.

After video.php is loaded, one of the tags is going to be clicked, a check is done to assure the video jumps to the right position and plays.

Test Case 2:

See Selenium source

This test case tests if the tags are loaded succesfully.

After video.php is loaded, we assert the elements of the tags on the page if they are present or not

Test Case 3:

See Selenium source

This test case tests if the rating system is working properly.

After video.php is loaded, the rating of a tag is set, a check is done to assure the tag rating is changed.

Test Case 4:

See Selenium source

This test case tests if the commenting system works.
After video.php is loaded, we first assert if the necessary elements are present (input comment, commentbutton).
After asserting the elements for their presence, we fill in the comment, and post it to the website.
A last check is done to verify the comment is posted.

Test Case 5:

See Selenium source

This test case tests if the tweetpoints work.
After video.php is loaded, we first assert if the necessary elements are present (tweetpoints).
After asserting the elements for their presence, we click the tweetpoint.
A last check is done to verify if the video jumps to the right place.

Test Case 6:

See Selenium source

This test case tests if the loginsystem works.
We first assert if the necessary elements are present (login-email, login-password, login-button, radiobuttons).
After asserting the elements for their presence, we login using a e-mail and password.
A last check is done to verify if the user has logged in.
Note:
The login system only works with the correct local database.
You can also register and change the values in Selenium to the new username and password.


Help:

These tests are made with the use of Selenium IDE. The HTML pages can be saved and openened in Selenium and run as tests. http://docs.seleniumhq.org/download/
